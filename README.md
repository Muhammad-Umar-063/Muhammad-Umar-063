### DTE GES, Command Center

Institutional governance dashboard for a CPA firm. Engagements move through a governed multi state workflow, every action writes an append only audit row, and an approval gate blocks a review from advancing without a formal CPA sign off. Realtime updates across sessions with optional Claude drafted delivery packets.
